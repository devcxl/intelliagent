## 🎯 使用入口

```
1. 命令行模式:
   python main.py "任务描述"

2. 交互式模式:
   python example.py

3. Python API:
   from main import IntelliAgent
   agent = IntelliAgent()
   agent.run("任务")

4. 运行测试:
   pytest test_intelliagent.py -v
```

## 🔧 配置方式

```
环境变量 (.env):
├── OPENAI_API_KEY        # OpenAI API 密钥
├── OPENAI_MODEL          # 模型选择
├── MAX_PDCA_CYCLES       # 最大循环次数（默认3）
├── MAX_RETRY_PER_STEP    # 最大重试次数（默认3）
└── EXPERIENCE_FILE       # 经验保存文件
```

## 💾 数据持久化

```
experiences.json
└── 保存任务执行经验
    ├── task: 任务描述
    ├── plan: 执行计划
    ├── results: 执行结果
    ├── status: 最终状态
    ├── score: 平均得分
    └── timestamp: 时间戳
```
