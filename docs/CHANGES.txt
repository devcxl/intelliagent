4. 功能变更
-----------
- 界面布局：从单页面垂直布局改为三栏布局
- 会话管理：新增会话创建、切换、删除功能
- 实时通信：使用 WebSocket 进行实时日志推送
- 组件化：使用 React + shadcn/ui 组件库
- 样式管理：使用 Tailwind CSS 进行样式管理
- 类型安全：全面使用 TypeScript
- 开发体验：支持热更新（开发模式）

5. 向后兼容
-----------
- 保留旧的静态文件在 web/static/ 目录
- 通过 WEB_ENV 环境变量切换模式
- 旧界面在开发模式下仍可使用

6. 使用方式
-----------
验证环境：
  ./verify-web.sh

生产模式：
  ./start-web.sh

开发模式：
  ./start-web-dev.sh

手动启动：
  WEB_ENV=production python web/server.py

7. 环境变量
-----------
WEB_ENV: production / development
WEB_HOST: 0.0.0.0 (默认)
WEB_PORT: 8000 (默认)
